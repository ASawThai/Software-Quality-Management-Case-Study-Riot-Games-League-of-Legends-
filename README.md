# Software-Quality-Management-Case-Study-Riot-Games-League-of-Legends-
This project analyzes software quality management practices at Riot Games.
The focus system is League of Legends, a live-service online game.

## Contents
- Company Background
- Quality Model (ISO 25010)
- Defect Analysis
- Risk Management
- CMMI Process Assessment
- GitHub Project Board Integration
